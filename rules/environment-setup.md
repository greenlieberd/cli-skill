# Environment Setup

## Prerequisites

- **macOS** — these patterns use Bun's `.env` loader and `import.meta.dir`
- **Bun ≥ 1.0**: `bun --version` (install: `curl -fsSL https://bun.sh/install | bash`)
- **`src/configure.ts`** in project (generated by `/new-cli`)

---

## configure.ts — copy verbatim into every project

```typescript
// src/configure.ts
import { readFileSync, existsSync } from 'fs'
import { join } from 'path'

export function loadEnv(): void {
  const path = join(import.meta.dir, '..', '.env')
  if (!existsSync(path)) return
  for (const line of readFileSync(path, 'utf8').split('\n')) {
    const eq = line.indexOf('=')
    if (eq < 1) continue
    const key = line.slice(0, eq).trim()
    const val = line.slice(eq + 1).trim().replace(/^["']|["']$/g, '')  // strip quotes
    if (key && !key.startsWith('#') && !process.env[key]) process.env[key] = val
  }
}

export function maskValue(v: string | undefined): string {
  if (!v) return '(not set)'
  if (v.length <= 8) return '••••••••'
  return v.slice(0, 4) + '•'.repeat(Math.min(v.length - 8, 16)) + v.slice(-4)
}
```

`loadEnv()` must be the first call in `src/cli.ts` — before any `process.env` access.

## .env.example — document every key

```bash
# .env.example — copy to .env and fill in values
# All keys are required unless marked (optional)

# Anthropic — get at console.anthropic.com
ANTHROPIC_API_KEY=sk-ant-...

# Firecrawl — get at firecrawl.dev (optional)
# FIRECRAWL_API_KEY=fc-...

# Reddit — no key needed for public search, key needed for higher limits (optional)
# REDDIT_CLIENT_ID=
# REDDIT_CLIENT_SECRET=
```

Rules:
- Every required key has a comment explaining where to get it
- Optional keys are commented out (prefixed with `#`)
- No actual keys ever — this file is committed

## Key validation on startup

Check required keys exist before running. Show a clear error, not a crash:

```typescript
// src/configure.ts — add this
const REQUIRED_KEYS: Array<{ key: string; hint: string }> = [
  { key: 'ANTHROPIC_API_KEY', hint: 'Get at console.anthropic.com → API Keys' },
]

export function validateEnv(): string[] {
  return REQUIRED_KEYS
    .filter(({ key }) => !process.env[key])
    .map(({ key, hint }) => `${key} is missing — ${hint}`)
}
```

```typescript
// src/cli.ts
import { loadEnv, validateEnv } from './configure.ts'
loadEnv()

const missing = validateEnv()
if (missing.length) {
  missing.forEach(msg => process.stderr.write(`  ✗  ${msg}\n`))
  process.stderr.write(`\n  Add these to .env — see .env.example\n`)
  process.exit(2)
}
```

## Display env status in config screen

```typescript
// In HUD config screen
const keys = [
  { name: 'ANTHROPIC_API_KEY', label: 'Anthropic' },
  { name: 'FIRECRAWL_API_KEY', label: 'Firecrawl (optional)' },
]

keys.forEach(({ name, label }) => {
  const val   = process.env[name]
  const status = val ? `\x1b[32m${maskValue(val)}\x1b[0m` : '\x1b[31m(not set)\x1b[0m'
  process.stdout.write(`  ${label.padEnd(22)}${status}\n`)
})
```

## Multi-environment support (optional)

For tools that need staging vs production:

```typescript
// Loads .env.local first (gitignored), then .env (committed defaults)
export function loadEnv(): void {
  for (const file of ['.env.local', '.env']) {
    const path = join(import.meta.dir, '..', file)
    if (!existsSync(path)) continue
    for (const line of readFileSync(path, 'utf8').split('\n')) {
      const eq = line.indexOf('=')
      if (eq < 1 || line.startsWith('#')) continue
      const key = line.slice(0, eq).trim()
      const val = line.slice(eq + 1).trim().replace(/^["']|["']$/g, '')
      if (key && !process.env[key]) process.env[key] = val
    }
  }
}
```

`.env.local` overrides `.env` — gitignore `.env.local`, commit `.env` with safe defaults.
